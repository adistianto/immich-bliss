# Downloads

Ready-to-paste **Custom CSS** files for Immich (**Administration → Settings → Custom Styling**).

## Files

**Default profile**

1. `immich-bliss-system.css` — recommended default
2. `immich-bliss-inter.css`
3. `immich-bliss-zalando.css`
4. `immich-bliss-gabarito.css`

**Max-performance profile** (disables frosted `backdrop-filter` tiers; same fonts)

- `immich-bliss-system-perf.css`
- `immich-bliss-inter-perf.css`
- `immich-bliss-zalando-perf.css`
- `immich-bliss-gabarito-perf.css`

Regenerate every file from source at the repo root:

```bash
npm run build:downloads
```

## How to use

1. Open one file.
2. Copy all content.
3. Paste into Immich **Custom CSS** and save.
